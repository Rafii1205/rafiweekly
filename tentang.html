About

I am an Information Technology student with a strong interest in Linux system administration, DevOps practices, and cloud infrastructure. I enjoy learning how modern systems are designed, deployed, monitored, and maintained to ensure reliability and performance. My learning journey focuses on hands-on practice, especially in working with Linux servers, containerization using Docker, and service orchestration concepts. I have built several small-scale infrastructure projects, including server monitoring using Prometheus, Node Exporter, and Grafana, as well as container-based load balancing using NGINX. Through these projects, I have gained practical experience in monitoring system resources, understanding traffic distribution strategies, and designing simple yet scalable architectures. I am particularly interested in how monitoring and automation can help detect issues early and improve system stability.
